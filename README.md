# tiptap-excalidraw-extension

excalidraw extension for tiptap

## Installation

```bash
# npm
npm install tiptap-excalidraw-extension

# yarn
yarn add tiptap-excalidraw-extension

# pnpm
pnpm install tiptap-excalidraw-extension
```
